3D Image Reconstruction of a Human Torso

For better and more complete documentation, go to: https://github.com/luantran/CapstoneProject/ This is where the open-source code for this software is located.

Loading the modalities

Before starting you must load the three modalities into the software. CLicking the first three buttons will allow you to do so. The text next to each button will confirm if you loaded the three modalities successfully.

Loading the landmarks

Next, you will need to load the landmarks. Note that these do not need to be done in order.
